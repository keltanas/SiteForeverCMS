-- ПРАВА ПОЛЬЗОВАТЕЛЕЙ --

Нужно создать таблицу групп пользователей
Каждого пользователь может состоять в некотором количество групп

Права раздаются аналогично ЮНИКС системам.

На какую-либо страницу или материал нужно хранить id владельца.

Можно задать следующие ограничения:

- Доступ на чтение, редактирование, создание владельцу ( применяется, если пользователь владелец )
- Доступ на чтение, редактирование, создание группе ( применяется, если группа пользователя совпадает с группой данных )
- Доступ на чтение, редактирование, создание гостям	- всем ( применяется, если не совпали предыдущие правила )
